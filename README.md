# learning-rust

## Install Rust

```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

```
source "$HOME/.cargo/env"
```

```
rustc --version
```

Compiler 
```
rustc intro/hello_world.rs 
```

Cargo 
```
cargo --version
```

Create new App
```
cargo new hello_cargo
```

Create new lib
```
cargo new hello_cargo --lib
```
 

```
cargo check
```

```
cargo run
```

```
cargo build --release
```
